## 🚀 About the Project

**Gamified Task Manager** is a modern productivity application that transforms everyday tasks into a **game-like experience**.

The project is being evolved into a **pixelated dungeon-style interface**, where users progress through a virtual world by completing real-life tasks.

🎯 Every action matters:

* Complete tasks → Gain XP
* Level up → Unlock progress
* Stay consistent → Build streaks

To make the experience even more engaging, the app introduces **AI-powered interactions**:

* 🤖 Smart **AI reminders**
* ✨ Motivational **AI compliments**

---

## 🎮 Core Features

* ✅ Task Management (Add / Complete Tasks)
* 🎯 XP Reward System
* 📈 Level Progression
* 📊 Game-style Progress Bar
* ✔ Interactive Tick System (click → turns green)
* ✨ Smooth animations & transitions
* 📦 Clean boxed UI layout
